Mosaic Editor
=============

### Installation

**Prerequisites**
* [Node.js](https://nodejs.org/)
* [Git](http://git-scm.com/)

**Install dependencies**

    npm install

### Development

    npm run dev

## Testing

    $ npm run test
    $ npm run test:unit
    $ npm run test:e2e

If chromedriver could not be found, make sure to update with:

    ./node_modules/protractor/bin/webdriver-manager update
